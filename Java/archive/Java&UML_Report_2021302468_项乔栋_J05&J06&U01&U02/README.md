## U02

### 类的定义及类的三要素

类描述了具有相同特性、行为、关系类别以及语义的一组对象，对象是类的一个实例。

类的三要素：属性，操作，关系。

### 类的可见性及 UML 的表示方法

- `+` Public 公有
- `#` Protected 受保护
- `-` Private 私有
- `~` Package 包内可见

> 在不同的 UML 建模软件中，可见性的实际渲染不尽相同，但用于描述可见性的符号保持一致。
